# 여행 운영 메모

## 정해야 할 것

- 여행 이름
- 여행 기간
- 방문 도시
- 멤버 이름 네 명
- 관리자 이름
- 기본 일정 원본
- 후보 장소 카테고리
- 관리자 승인 기준

## 기본 일정 관리 규칙

- 예약이 있는 일정은 시간과 링크를 반드시 입력한다.
- 시간이 정확하지 않으면 오전, 점심, 오후, 저녁처럼 시간대로 적는다.
- 식당/카페 후보는 기본 일정에 바로 넣지 않고 후보 장소로 관리한다.
- 기본 일정은 관리자가 수정한다.
- 당일 변경 사항은 일정 메모에 남긴다.
- 늦어질 가능성이 큰 날은 이동 시간을 넉넉히 잡는다.

## 후보 장소 규칙

- 장소명과 구글 지도 링크는 필수로 입력한다.
- 카테고리는 식당, 카페, 디저트, 바, 관광, 쇼핑, 기타 중 하나로 둔다.
- 추천 이유는 짧게 적는다.
- 가고 싶은 날짜나 관련 일정이 있으면 연결한다.
- 추천은 한 사람당 한 번만 누를 수 있다.
- 코멘트에는 예약 필요 여부, 웨이팅, 가격대, 영업시간 같은 정보를 남긴다.

## 관리자 승인 기준 초안

- 기본 일정과 동선이 맞는지 확인한다.
- 추천 수와 코멘트를 참고한다.
- 영업시간과 예약 필요 여부를 확인한다.
- 하루 일정이 과하게 빡빡해지지 않게 한다.
- 승인한 장소는 관련 날짜나 일정에 연결한다.
- 애매하면 보류로 두고, 이번 여행에서 제외하면 거절로 둔다.

## 중요 정보 체크

- 항공권
- 숙소 예약
- 기차/버스 예약
- 미술관/투어 티켓
- 여행자 보험
- eSIM 또는 로밍
- 현지 긴급 연락처

## 앱에 넣지 않는 것이 좋은 정보

- 여권 전체 번호
- 카드 전체 번호
- 주민등록번호
- 계정 비밀번호
- 민감한 개인 의료 정보
