## 접근 방식

- 지환, 지완, 지원은 이름을 누르면 바로 들어간다.
- 주환은 이름을 누른 뒤 관리자 접근 코드를 입력한다.
- 관리자 접근 코드는 문서에 직접 저장하지 않고 배포 환경변수 `ADMIN_ACCESS_CODE`로 관리한다.
- 주환만 후보 승인, 기본 일정 수정, 방문 완료 처리를 할 수 있다.
- 추천, 댓글, 후보 추가, 회계 입력은 네 명 모두 가능하다.

## 회계 규칙

- 각자는 자기 이름으로 들어와 비용을 기록한다.
- 비용은 `나에게 쓴 돈`과 `모두에게 쓴 돈`으로 나눈다.
- `나에게 쓴 돈`은 개인 기록용이며 정산 계산에 넣지 않는다.
- `모두에게 쓴 돈`은 네 명 공동 비용으로 보고 앱에서 1/N 계산한다.
- 앱은 공동 비용을 기준으로 누가 누구에게 얼마를 줄지 계산한다.
- 정산 완료는 주환, 지환, 지완, 지원 네 명 모두 눌러야 처리된다.
- 네 명 모두 정산 완료를 누르면 현재 공동 비용 정산 라운드가 닫히고 새 라운드가 0부터 시작된다.
- 개인 비용 기록은 정산 라운드 초기화와 무관하게 유지한다.
- 통화는 유로와 원화를 모두 지원한다.
- 앱에서 환율을 자동으로 가져와 유로/원화 환산 금액을 함께 보여준다.
