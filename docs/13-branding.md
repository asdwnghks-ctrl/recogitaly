# 브랜딩과 타이틀

## 앱 이름

이탈리아 가아즈아

## 이름의 의미

네 명의 이름인 주환, 지환, 지완, 지원에는 모두 `ㅇ`과 `ㅈ` 소리가 들어간다. 앱 타이틀에서는 `가아즈아` 중 첫 번째 `아`의 `ㅇ`과 `즈`의 `ㅈ`만 포인트 색상으로 강조한다.

## 기본 표기

일반 텍스트나 접근성 라벨에서는 항상 완성형 문장으로 쓴다.

```text
이탈리아 가아즈아
```

## 로고 타이틀 규칙

- 기본 글자 색은 진한 먹색 계열을 사용한다.
- 로고 폰트는 `Bagel Fat One`을 우선 사용한다.
- `가아즈아` 중 첫 번째 `아`의 `ㅇ`만 포인트 색상으로 표시한다.
- `즈`의 `ㅈ`만 포인트 색상으로 표시한다.
- 나머지 글자는 기본 글자 색으로 둔다.
- 접근성 텍스트는 `이탈리아 가아즈아`로 제공한다.

## 톤앤매너

방향은 `귀엽고 장난기 있는 단톡방 여행앱`이다.

- 친구들끼리 쓰는 말맛을 살린다.
- 문구는 짧고 명확하게 쓴다.
- 여행 중 바로 눌러야 하는 버튼은 장난스럽게 꼬지 않는다.
- 일정, 회계, 승인 같은 핵심 정보는 실용적으로 보여준다.
- 상태 문구는 가볍지만 헷갈리지 않게 쓴다.

예시:

| 상황 | 문구 |
| --- | --- |
| 오늘 일정 | 오늘도 가아즈아 |
| 후보 추가 | 여기 어때? |
| 승인 대기 | 픽스 기다리는 중 |
| 승인 완료 | 여기로 가아즈아 |
| 정산 필요 | 돈 계산 가아즈아 |
| 정산 완료 | 정산 클리어 |

## 색상

| 요소 | 색상 이름 | 색 |
| --- | --- | --- |
| 앱 배경 | White | `#ffffff` |
| 기본 글자 | Ink | `#1f2937` |
| `ㅇ` 포인트 | Blue | `#2563eb` |
| `ㅈ` 포인트 | Rose | `#e11d48` |
| 보조 경계선 | Line | `#e5e7eb` |
| 부드러운 표면 | Soft | `#f9fafb` |

## 폰트

| 용도 | 폰트 | 메모 |
| --- | --- | --- |
| 로고/앱 타이틀 | `Bagel Fat One` | 장난기 있는 앱 이름에 사용 |
| 본문/UI | `Pretendard` | 모바일 가독성 우선 |
| 숫자/시간/금액 | `Pretendard` | 필요하면 `font-weight: 600` |

## 구현 메모

한글 완성형 글자 안의 자음만 따로 색을 주는 것은 일반 텍스트 스타일링으로는 어렵다. 따라서 앱에서는 타이틀을 일반 텍스트가 아니라 `LogoTitle` 컴포넌트 또는 SVG로 만든다.

권장 방식:

- 화면 리더용 `aria-label`은 `이탈리아 가아즈아`로 둔다.
- 시각적 타이틀은 커스텀 컴포넌트로 그린다.
- 로고 컴포넌트의 전체 글자 분위기는 `Bagel Fat One`에 맞춘다.
- 첫 번째 `아`는 `ㅇ`과 `ㅏ`를 조합해 보이게 만들고, `ㅇ`만 Blue로 칠한다.
- `즈`는 `ㅈ`과 `ㅡ`를 조합해 보이게 만들고, `ㅈ`만 Rose로 칠한다.
- 작은 화면에서는 일반 텍스트 `이탈리아 가아즈아`로 fallback할 수 있다.

예상 구조:

```tsx
<LogoTitle ariaLabel="이탈리아 가아즈아">
  <span>이탈리아 가</span>
  <HangulSyllable initial="ㅇ" vowel="ㅏ" initialColor="#2563eb" />
  <HangulSyllable initial="ㅈ" vowel="ㅡ" initialColor="#e11d48" />
  <span>아</span>
</LogoTitle>
```

## 사용 위치

- 진입 화면 상단
- 홈 화면 상단
- 공유 링크 미리보기 제목
- 브라우저 타이틀
